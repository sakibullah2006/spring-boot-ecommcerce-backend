## Image Guidelines

- **Format**: PNG (recommended for screenshots) or PDF (for diagrams)
- **Resolution**: At least 1920x1080 for screenshots
- **Naming**: Use lowercase with hyphens (e.g., `architecture-diagram.png`)
- **Size**: Keep file sizes reasonable (< 5MB per image)

## Note

This directory is tracked by git. Large image files may need to be added to `.gitignore` or use Git LFS.
