## Technology Stack

- **Framework**: Spring Boot 3.3.5
- **Language**: Java 21
- **Database**: MySQL/MariaDB 5.5
- **Migration**: Flyway 10.10.0
- **Security**: Spring Security (Session-based)
- **Build Tool**: Maven
- **Testing**: PowerShell automation scripts
